欢迎使用店策 Agent

你的抖店和巨量千川经营副驾已安装成功。按照以下 3 步完成设置，即可开始使用。

Chrome 116+Edge
1

启动本地 Agent

本地 Agent 负责接收脱敏数据、生成建议和报告。首次使用需手动启动一次，之后会随系统自动启动。

  • 双击项目目录下的 bridge/enable_autostart.bat
  • 启动后访问 http://127.0.0.1:8765/health 确认正常
2

登录抖店和千川

在浏览器中打开并登录抖店和巨量千川后台。扩展只读取已登录页面的可见数据，不会保存账号密码。

3

固定扩展并首次巡查

将店策 Agent 固定到工具栏，点击扩展图标 → 自动获取全店数据。系统会依次打开 18 个核心页面采集数据。

  • 点击浏览器工具栏的拼图图标找到店策 Agent
  • 点击图钉按钮将其固定在工具栏
  • 点击扩展图标，点击自动获取全店数据
  • 巡检完成后打开经营副驾侧边栏查看建议
